#redux

Redux 简介

Redux 是 JavaScript 状态容器，提供可预测化的状态管理,采用了发布订阅设计模式

一些概念

store

Redux 的 createStore 方法创建 store，整个应用只有一个 store

state

store 维护的状态，本质是一个 js 普通对象

action

行为，本质是 js 普通对象，用来描述行为，它必须有一个 type 属性，type 属性在 reducer 处理时要用，除了 type 外你可以携带其他属性数据，提供给 reducer

dispatch

派发 action的方法，将action派发给 reducer 处理

reducer

处理 dispatch 派发过来的 action，处理后返回的结果 store 会将其替换成当前 state

7. isPlainObject.js

判断值是不是普通 js 对象
普通对象指的是字面量创建 let obj = {}，或者是构造函数 Object new 出来的 let obj = new Object()

核心思想：原型链

普通都是构造函数 Object new 出来的实例对象,而 Object new 出来的实例对象的原型指针__proto__都指向 Object 的 prototype,最终 Object.prototype 的__proto__指向 null，这就是原型链（如下图包括整个原型链的解释，所有的实例对象都指向构造函数的 prototype,所有的构造函数的 prototype 都指向 Object.prototype,最终指向 null）

发布订阅模式（也叫观察者模式）

它定义对象间的一种一对多的依赖关系当一个对象状态发生改变时，所有依赖它的对象都将得到通知
