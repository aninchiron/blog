Redux简介

Redux 是 JavaScript 状态容器，提供可预测化的状态管理

使用redux的createStore，创建一个store（一个应用建议只有一个store）。
store中维护了一个唯一的state（一个js对象）和订阅的函数队列listeners，
state是只读的，唯一改变state的方法就是使用store的dispatch，派发一个action，action经相应的纯函数reducer处理返回一个新的state替换现有的state，订阅者可以使用store的subscribe注册订阅函数，当reducer处理完成后，便会遍历所有的订阅函数并运行，通知state改变,从而所有的订阅者都知道state改变了，然后使用store的getState获取最新的state，并作出相应的处理

主要api

createStore, combineReducers, bindActionCreators,applyMiddleware, compose

它定义对象间的一种一对多的依赖关系当一个对象状态发生改变时，所有依赖它的对象都将得到通知
